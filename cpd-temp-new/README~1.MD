# Teaching about Grenfell: course access version

This folder contains a browser-based version of the course made available following the shutdown of the previous course platform. It is intended to keep the course accessible while a longer-term platform is arranged.

## What is included

- Course home page and module navigation.
- Main course pages converted to HTML.
- Local images, audio files and downloadable resources.
- Course file links rewritten to local links.
- Reflective quiz/survey activities converted into self-check or reflection pages.

## Current limitations

This version does not provide platform functions such as learner logins, submitted reflections, discussion forums, gradebook records, completion tracking or certificates. Reflective activities are for private note-making only; responses are not submitted or stored through this site.

The course will be hosted on a different platform as soon as possible.

## Quick deployment on GitHub Pages

1. Create a new GitHub repository, for example `grenfell-cpd`.
2. Upload the contents of this folder to the root of the repository.
3. Go to **Settings → Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select `main` and `/root`, then save.
6. GitHub will publish the site at a URL like `https://YOUR-USERNAME.github.io/grenfell-cpd/`.

## Using an existing website repository

If your existing website is already on GitHub Pages, you can place this whole folder inside a `cpd/` folder in that repository. The site should then be available at something like `/cpd/`. The links are relative, so it should work from a subfolder.

## Analytics

No external analytics script is included by default. If analytics are added later, provide an appropriate privacy/cookie notice and avoid collecting personal or sensitive learner data through the public site.
